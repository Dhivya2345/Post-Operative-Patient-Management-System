# Post-Operative-Patient-Management-System
Overview

This project is a web-based application designed to manage post-operative patient records in a digital and structured manner. It helps doctors and nurses efficiently store and update patient information, while providing interns with read-only access for academic learning purposes.

The system ensures secure data handling through role-based access control and centralized storage.

Features

Digital management of post-operative patient records

Role-based access for doctors, nurses, and interns

Secure authentication and data storage

Easy retrieval and update of patient information

Tech Stack

Frontend: HTML, CSS, JavaScript, React

Backend: Node.js

Database & Services: Supabase, SQL

Conclusion

The system improves efficiency in managing post-operative patient data and serves as a practical academic tool for understanding healthcare information systems.
